Usage
-----

Run the following to (re-generate the topics), which are save under a new `arxiv` folder in this directory (note, this will not be versioned). Make sure that your `PYTHONPATH` contains the path to this repository (`/full/path/to/Pivot/`) and that you have installed the `requirements.txt` for this repository (best doing this in a fresh environment).

```base

python arxiv_topics.py
```
